Personal Favorites Manager App

A simple, menu-driven Python application that allows users to store and manage personal preferences such as favorite fruit, color, and city.
Data is saved in a JSON file, making it persistent and reusable.

Features
1. Add New Favorite

Users can store:

Name

Favorite fruit

Favorite color

Favorite city

Each new entry is saved automatically to favorites.json.

2. View All Favorites

Display all saved records in a clean, readable format.

3. Search by Name

Find a specific person's saved favorites.

4. Data Persistence

All records are stored in favorites.json and automatically loaded when the program starts.

5. User-Friendly Menu Loop

The interface shows a menu repeatedly until the user chooses to exit.
